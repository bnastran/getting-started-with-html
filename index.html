HTML


HTML is abbreviation for Hypertext Markup Language. HTML documents are hearth of the web. it consist from text content – what you see, and markup, what it looks like. Also can have references to other documents like images or video, and has links to other documents


HTML markup


HTML markup basic elements are Tags. It has following structure <name> content </name> First one is called opening tag At the end we have closing tag Whole structure with opening tag, content and closing tag is ELEMENT.
